### **Mon Apr 14 — Nara Day Trip**
**Route (best for Nara Park side)**
1) Midosuji: **Shin-Osaka → Namba**
2) Kintetsu: **Osaka-Namba → Kintetsu-Nara**
3) Return same way

**Transit durations (round trip)**
- Midosuji: **~15 + ~5 wait** each way
- Kintetsu: **~40 + ~10 wait** each way
**Transit total:** ~**2h 20m** (excluding time in Nara)

**Notes**
- Kintetsu-Nara is walkable to Nara Park/Todaiji; it’s a good choice for the “park side” sights.
- Add 1–2 “anchors” so the day has shape (e.g., **Todaiji → Nara Park deer → Mochi/pastry stop**).

---

### **Tue Apr 15 — Kyoto Day Trip (Your Must-Visits)**
**Order (efficient + crowd-smart):** **Fushimi Inari → Gion → Nishiki Market**

1) Shinkansen: **Shin-Osaka → Kyoto**
2) JR Nara Line: **Kyoto Station → Inari Station** (Fushimi Inari)
3) Move **Inari → Gion**
   - Option A (rail): **Keihan** + short walks
   - Option B (taxi): fastest / simplest for 4 people
4) Move **Gion → Nishiki Market** (walk/taxi/bus depending on where you are)
5) Return: **Kyoto → Shin-Osaka** (Shinkansen)

**Transit durations (typical)**
- Shinkansen Shin-Osaka→Kyoto: **~15 min + ~10 wait**
- JR Kyoto→Inari: **~5 min + ~7 wait**
- Inari→Gion:
  - Rail option: **~12 min + ~6 wait** (+ walking)
  - Taxi option: **~20–30 min + ~5 wait**
- Return Kyoto→Shin-Osaka: **~15 min + ~10 wait**
**Transit total:** ~**1h 05m–1h 30m** (excluding sightseeing)

**Notes**
- Fushimi Inari is best **early**; consider an “arrive Inari before ~9:00” target if you want lighter crowds.
- For Shin-Osaka→Kyoto, Shinkansen is fastest, but there’s also a slower/cheaper **JR Kyoto Line** fallback if timing works.

- N’EX Shinagawa→NRT: **~75 min + ~20 wait**
- Hotel shuttle NRT→hotel: **~15–25 min + ~10–20 wait**
- Downtown shuttle hotel↔Narita Station (if used): **~15–25 min + ~10–20 wait** (each way)
**Transit total to hotel (from Osaka):** ~**4h** (excluding airport time)  
**Add Omotesando outing transit:** ~**1h** round trip (plus walking + dinner)

**Notes**

- Add a simple plan for the 4th person if only 3 go out (rest, onsen/relax, or meet later for dinner).
